Introduction
============

Policy product for QI TeamSpace integration on Plone 5, supercedes
the uu.projectsite, which provides the same for Plone 4.

Requires
--------

* Plone 5.0.6+

Integrates add-ons
------------------

* collective.teamwork

* wildcard.media

* collective.talkflow
